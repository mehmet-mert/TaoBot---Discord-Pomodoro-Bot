# TaoBot - Discord Pomodoro Bot

TaoBot çalışırken Pomodoro Tekniği ile zamanınızı yöneten bir Discord botudur. Yardım için `tao yardım` yazınız. Davet [linki](https://discord.com/api/oauth2/authorize?client_id=858212733820338198&permissions=3275776&scope=bot)

<img width=60% src="https://github.com/mehmet-mert/TaoBot---Discord-Pomodoro-Bot/blob/main/yardım.PNG">

## Başlarken

Repoyu klonlayın ve gerekli şeyleri kurun.

### Ön Koşullar

Ffmpeg bilgisayarınızda kurulu olmalı. Aşağıdaki komutu komut istemine yazarak kontrol edebilir, kurulu değilse [buradan indirebilirsiniz](https://www.ffmpeg.org/download.html).
```
ffmpeg
```

### Gerekli Modüller
Bu modülleri pip ile kurabilirsiniz.
```
pip install discord.py
pip install pafy
pip install PyNaCl
pip install youtube_dl
pip install opuslib
```

### Kurulum

1 - Depoyu klonlayın veya indirin.
2 - Ön Koşulları tamamlayın.
3 - Gerekli modülleri kurun.
